# Image Requirements

## Image Specifications

- **Format:** JPG (photos), PNG (logos/graphics), SVG (icons)
- **Max dimensions:** 1920px wide, 1080px tall
- **Compression:** Target < 200KB per image
- **Naming:** Lowercase, hyphens, descriptive (e.g., `venue-interior.jpg`)

## Photography Tips

1. **Lighting:** Natural light or well-lit indoor shots work best
2. **Action shots:** Try burst mode to capture mid-rally moments
3. **People:** Get permission before photographing members
4. **Building:** Include PCYC signage if visible
5. **Atmosphere:** Capture the friendly, casual vibe of sessions

## Optimization Process

1. Resize images to max 1920px wide
2. Compress using:
   - [TinyPNG](https://tinypng.com/)
   - [Squoosh](https://squoosh.app/)
   - [ImageOptim](https://imageoptim.com/) (Mac)
3. Target file size: <200KB
4. Consider WebP format with JPG fallback for better compression
