## SEO & Accessibility
- [ ] **Metadata:** Unique title and description for every page.
- [ ] **Semantic HTML:** Correct use of `<h1>`, `<h2>`, `<nav>`, `<main>`, `<footer>`.
- [ ] **Contrast:** Text meets WCAG AA contrast ratios (dark slate on white/light gray).
- [ ] **Performance:** Minimal external scripts (Google Fonts, GTM).
- [ ] **Links:** Internal linking structure connects key pages (Home -> Timetable -> Location).
